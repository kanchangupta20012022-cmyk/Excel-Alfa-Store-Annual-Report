# Alfa Store Annual Report 2021

##  Project Overview
This project is an interactive Excel dashboard analyzing annual sales performance.

##  Key Insights
- Total Sales
- Women Sales Share
- Orders Status Distribution
- Sales by Channel
- Top 5 Cities by Sales
- Age-wise Sales Distribution

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## File Included
- Annual Performance Dashboard (Excel)
